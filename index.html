Мои работы

HELPTENSE

Сервис для изучения английского языка

Задача

Разработка интерактивного веб-приложения для изучения английского языка разного уровня сложности для детей и взрослых.

Описание проекта

Сервис был разработан командой разработчиков из 4-х человек с использованием SCRUM. Моя роль: Front-end разработчик.

Реализация

Сервис включает в себя разделы: учебные материалы, словарь, интерактивный (обучение с помощью песен, обучение по фильмам и видео, тестирование, разговорник, перевод на время, проверка на знание слов), обучение по программам ( начальный уровень, средний, продвинутый). Также имеется личный профиль, в который входят: авторизация/регистрация, личный кабинет, мой словарь. Программы обучения позволят учащимся пройти всё обучение поэтапно и завершить закрепленные знания тестированием.

Инструменты

JS, Vue Js, HTML, CSS, SCSS, Github, Figma

Online Store

Интернет-магазин

Задача

Разработка онлайн-сервиса с возможностью заказывать товар из каталога. Необходимо было сверстать по макетам + разработать функционал frontend и backend (Node.js) частей + сборка проекта Webpack.

Реализация

Сервис состоит из нескольких страниц. Основной функционал в корзине: при нажатии на товар, каждая его единица добавляется в корзину, меняется стоимость, есть возможность редактирования списка покупок.

Инструменты

Vue.JS, JavaScript, HTML, CSS, Node.JS, Webpack, Zeplin, Figma, Github.

Personal finance app

Приложение для личных финансов

Задача

Разработка веб-приложения для контроля финансовых расходов.

Описание проекта

Веб-приложение, выполненное на Vue JS, которое позволяет наглядно видеть финансовые расходы. Внутри кода написан unit-тест.

Реализация

В веб-приложении есть возможность добавлять новые категории расходов, изменять стоимости, указывать конкретную дату, а также редактировать уже существующие. Диаграмма создана для визуальной наглядности расходов из списка. Сверху расположены уже готовые категории для ускорения процесса заполнения.

Инструменты

Vue Js, Vuetify, Vuex, Vue Test Utils, HTML, CSS, Github.
